Supported Features
Password Generator

Password Generator is a program designed to generate new passwords which includes random alphabets, numbers, symbols and special characters for the users. For this program, there are currently two functions available which includes generate (which generates the password) and save (which saves the password into its dedicated library). In addition, the users have the option to remove symbols or any special characters that are not needed.

Password Library

Password Library is a place in our website that is primarily designed to store and save the passwords that was made and generated previously. Password Library has a number of features such as quick-generating new passwords, deleting passwords and also an editing feature for convenience sake.

Password Strength Checker

Password Strength Checker is an additional program that can check how strong or weak your password is based on the length, combination of alphabets, numbers, and symbols. The password strength checker indicates the strength of the password by the strength bar. This function is provided as a way for users to check how strong their inputted passwords are.
